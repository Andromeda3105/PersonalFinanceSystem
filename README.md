# \# Personal Finance System (ІС КВК)

# \*\*Тема бакалаврської роботи:\*\* Інформаційна система обліку та аналізу персональних фінансів на основі компонентно-орієнтованої архітектури.

# \*\*Topic:\*\* Information system for personal finance accounting and analysis based on component-oriented architecture.

# 

# \## Опис проєкту

# Цей проєкт є вебзастосунком для управління персональними фінансами (PFM). Система розробляється для автоматизації обліку транзакцій, управління рахунками та забезпечення глибокого аналізу фінансового стану користувача за допомогою динамічної звітності.

# 

# \## Технологічний стек

# Згідно з архітектурним проєктуванням, система базується на сучасному стеку технологій:

# \* \*\*Бекенд:\*\* ASP.NET Core (C#)

# \* \*\*Фронтенд:\*\* Blazor WebAssembly

# \* \*\*База даних:\*\* PostgreSQL

# \* \*\*Архітектура:\*\* Clean Architecture

# 

# \## Основний функціонал

# \* Облік транзакцій (доходи, витрати, перекази).

# \* Управління фінансовими рахунками.

# \* Гнучка категоризація транзакцій.

# \* Динамічна звітність та аналітика.

# 

# \## Документування коду

# Усі публічні методи та моделі API задокументовані за допомогою XML-коментарів. Інструкцію з генерації та перегляду документації (OpenAPI / Swagger) дивіться у файлі \[docs/generate\_docs.md](docs/generate\_docs.md).
